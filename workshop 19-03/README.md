# Desafio Machine Learning

O objetivo foi criar uma classificação binária, capaz de identificar se um veiculo é elétrico ou não, baseando-se em indicadores de mercado e vendas globais da BMW.

O modelo atingiu uma acurácia de 72.0%, cumprindo o requisito de atingir uma performance de até 85%.

## Tecnologias
- Linguagem: python
- Biblioteca: Pandas, sklearn
